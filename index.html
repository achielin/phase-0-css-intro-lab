How does writing CSS differ from writing HTML

As we write CSS these are the type of questions we might ask ourselves:

  • Should we layout the text in a single or double column?
  • Should we use a different font colour for the header?
  • How should the very content appear on a desktop vs. a moile device?

All of the above questions deal with aesthetic and function considerations.These are perspectives of the presentational layer CSS.
Let's consider the type of questions we might ask ourselves as we write HTML:

  • Does the order of items within a list matter?
  • Should we wrap a list of links inside a navigation tag?
  • Is this the most important header in the whole HTML Document?

The last few questions deal with stracture, hierarchy and meaning.These are the perspectives of the content structure(HTML).
When we writ CSS we focus on aesthetic and function consideration.When we write HTML we focus on structure,hierarchy and meaning.
